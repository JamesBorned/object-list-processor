# object-list-processor

### Консольное приложение для обработки файлов с объектами

# Функционал:

* Прочитать список объектов из файла (файл лежит в той же директории, что и исходные файлы).
* Добавить объект в список.
* Группировать по расстоянию и сортировать по возрастанию расстоянию внутри каждой группы
(точка отсчёта - начало координат)
* Сохранить результаты группировки и сортировки в файл (файл в той же директории, что и исходные файлы.
* Выбор пункта меню. Отмена операции. Выход из программы.

# Нереализованный функционал (планируется):

* Сортировка по имени, типу, времени создания.
Код легко расширяется.

Файл решения открывается в Visual Studio.
